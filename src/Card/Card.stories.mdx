# Card
Cards contain content and actions about single subject.

# Usage
```js
import { Card, Button } from 'react-native-design-system';

<Card>
  <Title>Hello world</Title>
  <Button onPress={() => this.handleChange())}>
    Press Me
  </Button>
</Card>
```
