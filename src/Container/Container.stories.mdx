# Container
The container centers your content horizontally. It's the most basic layout element.
 
# Usage
```js
import { Container, Button } from 'react-native-design-system';

<Container>
  <Title>Hello world</Title>
  <Button onPress={() => this.handleChange())}>
    Press Me
  </Button>
</Container>
```
