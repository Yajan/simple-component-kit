# Title
Simple `Title` element that gives you different sizes and colors.

# Usage
```js
import { Title } from 'react-native-design-system';

<Title>
  Hello Design System
</Title>
```

# Props
<Props of={Title} />



# Size
Uses size defined in `fontSize` in theme.

<Preview>
  <Story name="size">
     
      <Title size={24}>
        Hello Design System
      </Title>
      
  </Story>
</Preview>

# Playground
Play with Title in Canvas mode.

<Story name="playground" parameters={{ decorators: [withKnobs] }}>

  >
